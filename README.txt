COACH TASCABILE v2 - ISTRUZIONI

Contenuto:
- scheda settimanale 5 giorni
- libreria esercizi con immagini schematiche per gli esercizi principali
- timer recupero da 45/60/90/120/180 secondi
- diario progressi con suggerimento automatico: aumenta peso / mantieni
- dieta di 14 giorni
- sezione misure corporee con grafico peso e vita

Uso su iPhone:
1. Carica la cartella su iCloud Drive, un piccolo hosting, o apri index.html da un server locale.
2. Apri index.html con Safari.
3. Tocca Condividi > Aggiungi alla schermata Home.

Uso su Android:
1. Apri index.html con Chrome, preferibilmente da un piccolo hosting o server locale.
2. Menu > Installa app / Aggiungi a schermata Home.

Nota pratica:
Aprire un file HTML direttamente dal telefono può limitare alcune funzioni PWA/offline per regole di sicurezza del browser. La soluzione più stabile è caricare la cartella su Netlify, GitHub Pages, iCloud/hosting web o usare un piccolo server locale.

Dati:
I dati di diario e misure vengono salvati in locale sul dispositivo tramite localStorage.
