COACH TASCABILE V14

Novità principali:
- aggiornate le infografiche degli esercizi agli anelli con nuove versioni in stile coerente con la guida del push-up;
- sostituite le immagini di Ring Row, Ring Push-Up e Support Hold agli Anelli;
- per l'esercizio combinato ring-pushup ora la guida mostra sia l'infografica del Push-Up classico sia quella del Ring Push-Up, così puoi scegliere quale variante eseguire;
- mantenute tutte le funzioni della V13 (focus braccia, reset settimanale serie, dieta aggiornata, PWA).

File/versioning:
- VERSION = '14'
- manifest e service worker aggiornati a ?v=14
- cache aggiornata a coach-tascabile-v14-20260625

Dati salvati:
- restano compatibili le stesse chiavi localStorage della V11/V12/V13.

Come aggiornare GitHub Pages:
1) Decomprimi coach_tascabile_v14.zip.
2) Entra nella cartella coach_tascabile_v14.
3) Carica su GitHub il contenuto interno della cartella, non la cartella intera.
4) index.html deve stare nella root del repository.
5) Dopo l’upload apri il sito con ?v=14 per evitare cache Safari.
6) Se Safari mostra ancora la vecchia versione, elimina i dati sito di github.io e riapri l’icona Home.
