COACH TASCABILE V13

Obiettivo della versione:
- mantenere tutte le funzioni della V12;
- aggiornare Upper A, Upper B e Upper C con focus braccia moderato;
- inserire lavoro diretto per bicipiti e tricipiti, con particolare attenzione al capo lungo del tricipite;
- introdurre esercizi agli anelli in modo progressivo e sicuro;
- aggiungere reset settimanale delle serie completate;
- aggiungere una colazione alternativa senza yogurt greco con latte parzialmente scremato.

- nuove immagini/infografiche create in assets:

- dieta mantenuta dalla V12 + colazione senza yogurt greco:
  latte parzialmente scremato 250 ml + whey 20–30 g + avena 35–45 g + frutta 150–200 g.
- versione/cache aggiornate a V13.

Come aggiornare GitHub Pages:
1) Decomprimi coach_tascabile_v13.zip.
2) Entra nella cartella coach_tascabile_v13.
3) Carica su GitHub il contenuto interno della cartella, non la cartella intera.
4) index.html deve stare nella root del repository.
5) Dopo l’upload apri il sito con ?v=13 per evitare cache Safari.
6) Se Safari mostra ancora la vecchia versione, elimina i dati sito di github.io e riapri l’icona Home.
